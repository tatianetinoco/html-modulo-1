O que são Clients e Servers ?

Servidor (Servers) é o computador no qual foi configurado em uma rede para oferecer
algum tipo de serviço. Já o cliente (Clients), é o computador que está do outro lado, que foi
configurado para utilizar o serviço que é oferecido pelo servidor. Neste caso, podemos
afirmar então que o computador servidor é aquele que oferece um serviço e o
computador cliente é aquele que utiliza um serviço. Por exemplo, ao acessarmos
uma página da web, a mesma é disponibilizada a partir de um computador servidor.
O nosso computador, chamado de cliente, irá realizar as solicitações de acesso às
páginas da web que estão armazenadas em um servidor. É importante ressaltar que os
computadores clientes também podem ser chamados de estações. Inicio


O que é HTML?

HTML significa HiperText Markup Language (Linguagem de Marcação de Hipertexto)
não é uma linguagem de programação, mas sim de marcação utilizada na construção de páginas
na Web para estruturar o conteúdo. A página HTML é composta por elementos definidos através de tags. Inicio


Como é uma estrutura básica de um HTML?

A estrutura básica de uma página HTML é formada pelas tags <html>, <head> e <body> e a instrução <!DOCTYPE>.
Indica como o conteúdo deve ser interpretado pelos navegadores. Inicio



Entendendo a diferença entre listas ordenadas e não ordenadas

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que
os itens de cada lista são apresentados - a lista ordenada enumera os elementos e a
lista não-ordenada coloca marcadores antes de cada elemento.
Uma lista ordenada é delimitada pelas etiquetas <ol> e </ol> (Ordered List).
Cada item da lista é delimitado pelas etiquetas <li> e </li> (List Item).
A estrutura da lista não-ordenada é semelhante a da lista ordenada - porém, ao invés
de ser construída com o elemento OL, é construída com o elemento UL (Unordered List).

ol = ordened list (lista numerada)
ul = unordened list (lista ordenada)
li = list item (item da lista)

Lista ordenada:

  1. Item número um.
  2. Item número dois.
  3. Item número três.
Lista não-ordenada:
  • Primeiro item.
  • Segundo item.
  • Terceiro item.

Criando links em HTML

A tag <a> pode ser utilizada para criar alguns tipos diferentes de links. Ela pode
ser utilizada para criar âncoras, links para dentro da página e links para páginas externas. É
preciso definir alguns atributos para o seu funcionamento. O atributo href é onde colocamos a
URL (endereço do link). Ao clicar nesse link o navegador vai carregar a página informada no href.

<a href="URL">Texto Exibido</a>

Para que o link abra em uma nova aba apenas defina o atributo target como _blank:

<a href="URL" target="_blank">Texto Exibido</a>


SEMPRE adicione rel="noreferrer noopener" ao elemento de âncora quando usar o atributo target:

<a href="URL" target="_blank" rel="noreferrer noopener">Texto Exibido</a>

O atributo rel define a relação entre sua página e o URL vinculado. Configurá-lo
como noopener noreferrer evita um tipo de phishing conhecido como 'tabnabbing'

Nota: Um URL pode apontar para arquivos HTML, arquivos de texto, imagens, documentos de texto,
arquivos de vídeo e áudio e qualquer outra coisa que possa estar na Web.
